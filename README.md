# Spluk API

Public REST API for [spluk.app](https://spluk.app) — anonymous messaging platform.

## Base URL
`https://api.spluk.app.`

## Documentation
- Interactive: https://api.spluk.app/openapi.yaml (view with Redoc/Swagger)
- Root: https://api.spluk.app

## Endpoints
- `POST /v1/messages/@username` → Send message
- `GET /v1/profiles/@username` → Public profile
- `GET /v1/stats/@username` → Message stats
- `GET /v1/world` → World chat
- `GET /v1/username-available?username=...`

Made with ❤️ for the Spluk community.
